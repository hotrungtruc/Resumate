# Resumate - AI Job Matching & CV Builder

![Project Status](https://img.shields.io/badge/status-MVP-success?style=flat-square)

> **Resumate** is an intelligent career companion designed to help students and job seekers tailor their resumes to specific job descriptions (JD) and identify skill gaps using AI.

## 🚀 Features (MVP)

* **📊 Interactive Dashboard:** Modern, dark-mode UI built with Tailwind CSS for tracking application status.
* **🎯 AI Matcher:** Analyze the compatibility between your CV and a specific Job Description (JD).
* **🧠 Skill Gap Analysis:** AI identifies missing skills and suggests a learning path.
* **📝 CV Management:** Store and manage multiple versions of your CVs.
* **kanban Job Tracking:** Track your applications through stages (Applied, Interview, Offer, Rejected).

---

## 🛠 Tech Stack

### Backend
* **Framework:** [FastAPI](https://fastapi.tiangolo.com/) 
* **Database:** PostgreSQL.
* **ORM:** SQLAlchemy / SQLModel.
* **AI Integration:** OpenAI API / Google Gemini API.

### Frontend
* **Core:** HTML
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) 
* **Icons:** Remix Icon.
